About the Application

DiaPredict is an AI-powered web application designed to predict the risk of diabetes using machine learning techniques. The system analyzes important health parameters such as age, blood glucose level, blood pressure, body mass index (BMI), pulse rate, and family medical history.

The main goal of DiaPredict is to support early detection and preventive healthcare. By identifying diabetes risk at an early stage, users can take timely medical consultation and make informed lifestyle changes to reduce long-term health complications.

How It Works

  1. User enters health details
  2. Data is sent to the FastAPI backend
  3. AI model analyzes the data
  4. Diabetes risk is predicted instantly
